# Wodi Design System

Authoritative source:

- `memory/wodi-design-system.html`

This file exists so repo instructions that reference `memory/wodi-design-system.md` stay valid. For implementation details, visual examples, CSS tokens, component rules, and agent handoff guidance, open the HTML handoff.

Core non-negotiables:

- Background: `#0c0d0f`
- Primary text: `#f2f0eb`
- Single accent: `#f5c200`
- Display font: Barlow Condensed
- Body/UI font: Barlow
- Avoid cyan, magenta, blue accents, text gradients, pure black `#000`, and pure white `#fff`
- Recap screens must show the full workout structure, actual result, and totals
